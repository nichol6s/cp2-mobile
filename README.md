## 🚀 Iniciando o projeto
 
```sh
#Clone o repositorio
$ git clone https://github.com/nichol6s/cp2-mobile
 
#Instale as dependências
$ npm install
 
#Inicie o app
$ npx expo start
```
 
## 💻 Funcionalidades
  - Cadastro de Items
  - Listagem de Items
  - Exclusão de Items
  - Favoritar Item
  - Excluir dos favoritos
  - Página de usuário
  - Navegação intuitiva com Expo Router
